= SnapManager host
:icons: font
:imagesdir: ../media/

[.lead]
The SnapManager host is a Windows server, which also runs other NetApp products.

The SnapManager host is installed with the following products:

* SnapDrive for Windows
* Host Utilities

The SnapManager host runs as a service.
